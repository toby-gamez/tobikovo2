Tato stránka se věnuje konstrukci trojúhelníků a rovnoběžníků podle různých zadání. Najdete zde postupy pro sestrojení trojúhelníků podle vět SSS, SUS, USU a dalších, stejně jako návody na konstrukci rovnoběžníků. Naučte se, jak správně postupovat při rýsování základních geometrických útvarů.

Trojúhelníky

Věta SSS

Mějme trojúhelník ABC, kde c = 5 cm cm, b = 3 cm a c = 4 cm.

  1. AB: |AB| = 6 cm
  2. k: k (A; r = 3 cm)
  3. l: l (B; r = 4 cm)
  4. C: C ∈ k ∩ l
  5. △ABC

Věta SUS

Mějme trojúhelník KLM, kde k = 3 cm, m = 5 cm, |∠ KLM| = 60°.

  1. KL: |KL| = 5 cm
  2. →LX: |∠ KLX| = 60˚
  3. k: k (L; r = 3 cm)
  4. M: M ∈ k ∩ →LX
  5. △KLM

Věta USU

Mějme trojúhelník EFG, g = 6 cm, |∠ EFG| = 45°, |∠ GEF| = 70°.

  1. EF: |EF| = 6 cm
  2. →FX: |∠ EFX| = 45°
  3. →EY: |∠ FEY| = 70°
  4. G: G ∈ →FX ∩ →EY
  5. △EFG

Strana, strana, výška

Mějme trojúhelník ABC, c = 4 cm, a = 3,5 cm, vc = 3 cm.

  1. AB: |AB| = 4 cm
  2. k: k (B, r = 3,5 cm)
  3. q: q // AB, |AB q| = 3 cm
  4. C: C ∈ q ∩ k
  5. △ABC

Strana, výška, těžnice

  1. AB: |AB| = 5 cm
  2. q: q // AB, |q AB| = 3 cm
  3. k: k (SAB, r = 4 cm)

Rovnoběžníky

Strana, úhel, výška

Mějme rovnoběžník ABCD, a = 3 cm, α = 30°, va = 3 cm.

  1. AB: |AB| = 3 cm
  2. →AX: |∠ BAX| = 30°
  3. →AY: |∠ ABY| = 150°
  4. p: p // AB, |p AB| = 3 cm
  5. C: C ∈ →AX ∩ p
  6. D: D ∈ →AY ∩ p
  7. ◆ABCD

Strana, strana, úhlopříčka

Mějme rovnoběžník ABCD, a = 4,5 cm, b = 3 cm, |AC| = 6 cm

  1. AB: |AB| = 4,5 cm
  2. k: k (B, 3 cm)
  3. k': k' (A, 3 cm)
  4. l: l (A, 6 cm)
  5. C: C ∈ k ∩ l
  6. p: p // AB, C ∈ p
  7. D: D ∈ p ∩ k'
  8. ◆ABCD
